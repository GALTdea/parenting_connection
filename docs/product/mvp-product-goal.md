# MVP Product Goal

## Goal

Ship a private Rails-first MVP that helps a parent create child profiles, build a meaningful daily conversation ritual, preserve text and voice memories, review a timeline, and receive gentle AI-assisted reflections.

## MVP Scope

The MVP includes:

- Parent accounts
- Child profiles
- Daily question system
- Prompt library
- Text responses
- Voice recordings
- Timeline view
- Memory archive
- Basic AI summaries
- Monthly reflection letter

## Primary Loop

1. Parent signs in.
2. Parent selects or creates a child profile.
3. Parent answers a daily question with text, voice, or both.
4. The response becomes part of the child's private timeline.
5. Parent can revisit the memory archive.
6. AI helps summarize patterns and prepare a monthly reflection letter.
7. Parent reviews AI-assisted content before saving, sending, or sharing.

## Success Definition

The MVP succeeds if a parent can consistently preserve meaningful child memories with low friction and feel that the app helps them notice their child more deeply over time.

## MVP Quality Bar

- Clear account and profile ownership
- Simple, mobile-friendly capture flow
- Private-by-default memory archive
- Gentle product copy
- Parent-reviewed AI summaries
- No diagnostic language
- No unnecessary collection of child data

## Out Of Scope Until After MVP

- Temperament profiles
- Personality graphs
- Advanced personalization
- Gifting
- Printed books
- Multiple family members
- Grandparent accounts
- Educational recommendations
- Child developmental assessments
- Social features
- Multi-household permissions
- Sharing outside the parent account
- Public links
- Paid subscriptions
- Native-only features
- Complex AI personalization
- Clinical, educational, or behavioral recommendations
